# 千章小说创作系统 — 使用说明

你好！这是一个开箱即用的 AI 自动写小说工具。
你不需要安装任何编程环境（如Python），双击 exe 即可运行。

## 📁 目录说明

### 核心文件
*   **`千章小说创作系统_桌面版.exe`**：程序的运行入口，**双击即可启动软件**。
*   **`gui_app.py`**：源码主程序（Python开发者用）。
*   **`generator.py`**：小说生成引擎核心模块。
*   **`rag_engine.py`**：RAG 智能记忆检索引擎（自动匹配最相关的设定）。
*   **`config.json`**：配置文件（API Key、模型名称、温度等）。
*   **`prompts.json`**：AI 提示词模板。

### 小说数据文件夹（每个项目一套）
*   **`characters/`**：角色设定（每个角色一个 .txt 文件）。
*   **`world_building/`**：世界观设定。
*   **`plot/`**：大纲、备忘录、基调铁律。
*   **`output/`**：AI 生成的最终小说章节。
*   **`history/`**：历史记录备份。

## 🚀 如何开始使用

1.  **填写 API 密钥**：
    *   启动软件后点击左上角「⚙ 设置」按钮。
    *   填入你的 DeepSeek API Key（必填）。
    *   如需切换 MiniMax 模型，同时填写 MiniMax API Key。
2.  **选择/创建项目文件夹**：
    *   首次启动会要求你选择小说项目文件夹。
    *   系统会自动创建所需的子文件夹（characters/ world_building/ plot/）。
3.  **准备设定**：
    *   将角色设定放入 `characters/` 文件夹（每个角色一个 .txt）。
    *   将世界观背景放入 `world_building/` 文件夹。
    *   将剧情大纲/细纲放入 `plot/` 文件夹。
    *   **【重要】** 将你的小说基调规则放入 `plot/基调铁律.txt`（系统会自动加载）。
4.  **开始创作**：
    *   在界面左侧勾选本章需要的设定（系统也会自动 RAG 检索最相关的）。
    *   在写作要求栏输入本章细纲，点击「📝 写新章」。

## 🛠 功能一览

| 按钮 | 功能 |
|------|------|
| 📝 写新章 | 根据细纲生成新章节 |
| 📎 续写 | 在已有章节基础上继续写 |
| 🚀 批量挂机写 | 全自动批量生成多章，含自动记忆压缩 |
| 🧠 提炼记忆 | 手动压缩全局备忘录 |
| 🔮 伏笔追踪 | AI 自动追踪未解伏笔 |
| ✨ 润色正文 | AI 优化文笔和节奏 |
| 🔍 逻辑检查 | AI 审查逻辑硬伤和人设崩塌 |
| 📜 进度编年史 | 自动推演时间轴并记录里程碑 |
| 📂 打开历史章节 | 浏览和加载旧章节 |
| 📑 一键导出排版 | 合并所有章节，自动中文排版导出 TXT |
| 右键选中文本 | ✨ AI划线重写/扩写（局部精修） |

## 💡 小贴士

*   **RAG 自动检索**：系统会根据你的写作要求自动从知识库中挑选最相关的设定投喂给AI，无需手动逐个勾选。
*   **基调铁律**：在 `plot/基调铁律.txt` 中写入你的小说基调规则（比如"禁止出现暴力犯罪元素"），系统会在每次生成时自动注入。换书的时候只需修改此文件。
*   **编年史**：每写完一章点一次「📜 进度编年史」，系统会自动维护一个全局时间轴文件 `plot/世界编年史.txt`。
*   **一键导出**：点击「📑 一键导出排版」，自动合并所有章节并套用标准中文首行缩进格式，可直接复制到各大小说平台。

如有问题，请检查你的 API 接口余额是否充足，或网络是否通畅。祝你创作愉快！
